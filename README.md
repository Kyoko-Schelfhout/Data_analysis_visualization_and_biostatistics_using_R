# Assignment from BIT08 – Walks, Weather & Weekdays: A Dog Walking Data Analysis
This repository contains an assignment completed for BIT08 – Data Analysis, Visualization and Biostatistics using R, part of my Bioinformatics studies at Howest University of Applied Sciences in Bruges, Belgium.


## 📋 About the Assignment
This report presents an exploratory data analysis of the WalkTheDogs dataset, this is a personal activity log of 223 days tracking daily step counts alongside contextual lifestyle variables. The dataset was sourced from the Stat2Data package, available via the R Datasets repository.
The central research question is: What factors are associated with how often the dogs were walked? Specifically, the analysis investigates whether dog walking frequency is influenced by weather conditions and day of the week.


## 📊 Visualizations Included
* Horizontal barplot: walked vs. non-walked days
* Lollipop chart: weather condition frequency
* Grouped barplot: walk frequency by day of the week
* Density plot: walk distance by weather and walk status
* Violin plot & boxplot: step distribution by weather condition
* Barplot: dog walk rate by weather
* Spider/Radar char: walk rate by day of week and weather


## 🛠️ Tools & Packages Used
R (R Markdown), with the following packages: ggplot2, dplyr, tidyr, tidyverse, MetBrewer, RColorBrewer, colorspace, fmsb
